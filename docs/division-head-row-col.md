# 구분 헤드(Division Head)에서의 행, 열 추가 및 삭제
- 구분 헤드도 컴포넌트 내에서는 하나의 블록으로 취급 (실제 블록이라는 것은 아님)
    - 해당 명세서에서는 편의상 "구분 헤드 블록"이라고 칭하겠음

## 열 추가
- 구분 헤드 블록에 새로운 열을 추가한다. (맨 오른쪽)
- 구분 헤드 블록에 N개의 열이 있다면, N+1 열은 N열의 병합 상태를 그대로 복사한다.

## 행 추가
- 선택된 셀 기준으로, 바로 아래에 행을 하나 추가한다.
- 행 추가 시, 구분 헤드뿐만 아니라 같은 컴포넌트에 속한 모두 블록의 행이 하나씩 추가된다.
    - 각 블록에 행 추가 시, 각 블록에 해당하는 셀 타입이 추가된다. 

### 구분 헤드에서의 행 추가
- (R,C), 즉 R행 C열을 기준으로 "행 추가"를 실행했을 경우,
    1. R+1 자리에 행이 하나 삽입된다. (해당 위치의 기존 행들은 아래로 밀린다)
    2. (R,C-1) 셀과 (R+1, C-1) 셀은 병합된다 (rowspan)
        - C = 0이라면, 병합되지 않고, 3번이 실행된다.
    3. 병합되지 않은 독립된 (R+1,C) 셀이 하나 추가된다. 
        - 구분 헤드 블록의 열 개수가 M개라면, (R+1,C), (R+1,C+1), (R+1,C+2), ... , (R+1, M-1) 셀이 추가된다.
